# ① js中处理异步函数的方法

> questions 是异步函数

## async/await 方式

使用 `await` 异步函数，需要在 `async` 函数中使用。

**特殊点**：允许在顶层使用 `await`，即不在函数里面。

`promises` 方法，对比于回调函数方法，更简洁。`async/await` 语法底层基于 `promises` 实现。

`await` 用于 `promises` 对象上。

```javascript
const n = await rl.question("请输入n值：");     
```

## 回调函数方式

```javascript
rl.question("请输入n值：", (answer) => {
    const n = Number(answer);
    console.log(generateMatrix(n));
    rl.close();
});
```

# ② 解构赋值

> 当得到的结果为字符串时，不要使用解构赋值，否则会报错

**错误写法**：

```javascript
const {n} = await rl.question("请输入n值：");  // ×
```

**正确写法**：

```javascript
const n = await rl.question("请输入n值：");  // ✓
```
