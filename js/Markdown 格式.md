# Markdown 语法笔记

## 1. 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 2. 引用块

```markdown
> 这是引用内容
> 可以多行
```

## 3. 分隔线

```markdown
***
---
___
```

三种写法效果相同，渲染为水平分隔线。

## 4. 文字样式

```markdown
**加粗**
*斜体*
***加粗斜体***
~~删除线~~
```

## 5. 行内代码

```markdown
`代码`
```

用于高亮关键词或简短代码。

## 6. 代码块

````markdown
```javascript
const n = 10;
console.log(n);
```
````

支持多种语言：`javascript`、`python`、`html`、`css` 等。

## 7. 列表

**无序列表**：
```markdown
- 项目1
- 项目2
  - 子项目
```

**有序列表**：
```markdown
1. 项目1
2. 项目2
```

## 8. 链接与图片

```markdown
[链接文字](https://example.com)
![图片描述](图片路径)
```

## 9. 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容 | 内容 | 内容 |
```

## 快捷操作

1. `Ctrl + Shift + V` - 打开 Markdown 预览
